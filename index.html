はじめに読まれる方へ

こちらのサイトは、Web系の言語と呼ばれている言語に特化して解説をしていくサイトです。主な内容は下記の通りです。

  • HTML（マークアップ言語）
  • CSS（スタイルシート言語）
  • JavaScript（プログラミング言語）

これらの言語はそれぞれWebページ制作に使われることが多い言語です。

  • HTML(Hyper Text Markup Languageの略)

    Webページを作成するに当たって、この言語がないと話になりません。この言語を使ってWebページの中の文字を記載していきます。なので、文字を入れなくてもよい！とか言う人がいれば・・・　いや、無理か。
    また、この言語はひとつひとつの言葉に、これは「見出し」、これは「リスト」といったことを書いていくことで、Webページを検索するときなどに機械が分かりやすいという利点もあるので、その場所にあったhtmlの記載が必要となります。

  • CSS(Cascading Style Sheetsの略)

    こちらは現在のWebページ作成をするに当たり、必須ですね。ただ、HTMLだけでも文字は翔ので、文字だけのWebページで良ければ、極論で言うと要らない言語になります。
    この言語は、Webページの見た目を作成するために使われる言語です。Webページに色を付けたり、文字のサイズを変えたり、画像や文章の場所を指定したりするための言語にあたります。

  • JavaScript

    この言語はWebページに動きをつけることができる言語です。なのでWebページがに動きがないサイトはこちらの言語は使っていないと思ってもらって良いかと思います。。つまりは動かないサイト（静的なサイトと呼ぶ）を作るのであれば、別段必要のない言語と言うことですね。
    この言語は、先述の通り動きをつける為の言語であり、○○があった時に、△△に対して、××をしてくださいという命令（プログム）を与えること（プログラミング）ができる言語であり、こういった命令を与えることができる言語をプログラミング言語と呼びます。これによって、Webページに動きを与えることができます。

それでは、ここから頑張って html と css と JavaScriptを勉強していきましょう。
